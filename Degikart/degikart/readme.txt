== Description ==
Degikart is a powerful and versatile digital marketplace theme designed for buying and selling products. It is ideal for creating e-commerce websites, online stores, and marketplaces. The theme is fully responsive and compatible with the latest WordPress versions.

== Features ==
- Customizable Layouts: Choose between one-column, two-columns, or three-columns layouts.
- Fully Responsive Design: The theme is mobile-friendly and looks great on all devices.
- Custom Header & Background Options: Customize the header and background easily from the WordPress Customizer.
- Translation Ready: The theme is translation-ready and supports multiple languages.
- Theme Options Panel: Customize the theme with ease using the built-in theme options panel.
- Sticky Post & Threaded Comments: Enable sticky posts and threaded comments for a dynamic blog experience.
- Footer Widgets: Add widgets to the footer to display additional information and links.
- Featured Images: Display beautiful featured images for your posts and pages.

== Installation ==
1. Download the theme ZIP file.
2. Go to your WordPress dashboard.
3. Navigate to **Appearance > Themes** and click **Add New**.
4. Click the **Upload Theme** button and choose the theme ZIP file.
5. Click **Activate** once the theme is uploaded.

== Theme Customization ==
After activating the theme, you can customize it via the **Appearance > Customize** section in the WordPress dashboard. Here, you can change various settings like:
- Site Title & Tagline
- Header Image
- Colors
- Layouts
- Widgets
- Menus
